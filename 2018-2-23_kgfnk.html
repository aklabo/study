  • junegunn/fzf： A command-line fuzzy finder

aliasを作っておくと便利かも

選択したmarkdownファイルをエディタで表示

% vim $(find . -type f -name "*.md" | peco)

選択したディレクトリへ変更
(カレント以下全ての階層が対象)

% cd $(find . -type d | peco)

プロセスを選択してkill

% kill -9 `ps ax | peco | awk '{print $1}'`

Gitブランチを選択してチェックアウト

% git checkout $(git branch -a | peco | head -n 1 | sed -e "s/^\*\s*//g")

組み合わせ次第で無限の可能性を感じます
